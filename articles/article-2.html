JavaScript Closures Explained

A deep dive into closures in JavaScript and how they work.

What is a Closure?

A closure is a function that retains access to its lexical scope, even when the function is executed outside that scope. In simpler terms, it "remembers" the environment in which it was created.

How Closures Work

Here’s an example of a closure in action:

function outerFunction() {
  let outerVariable = 'I am outside!';

  function innerFunction() {
    console.log(outerVariable); // Access outerVariable
  }

  return innerFunction;
}

const closureExample = outerFunction();
closureExample(); // Logs: "I am outside!"

Practical Uses of Closures

  • Data Encapsulation: Closures can be used to create private variables.
  • Callbacks: Closures are often used in asynchronous programming.
  • Function Factories: Closures can generate functions with specific behavior.
